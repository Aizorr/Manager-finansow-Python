# Personal Finance Manager (Python / OOP)

A command-line interface (CLI) application built with Python to help users manage, track, and analyze daily personal finances.

## Key Features
- **Object-Oriented Architecture:** Clean separation of concerns using Python classes and methods.
- **Data Persistence:** Reads and writes transaction records to local files (`.txt`/`.csv`).
- **Data Parsing & Calculations:** Automatically calculates totals, balances, and category breakdowns.
- **Robust Error Handling:** Input validation and custom exception handling to ensure application stability.

## Tech Stack & Tools
- **Language:** Python 3.x
- **Concepts:** Object-Oriented Programming (OOP), File I/O, Error Handling, Logic & Data Parsing
- **Tools:** VS Code / PyCharm, Git & GitHub
